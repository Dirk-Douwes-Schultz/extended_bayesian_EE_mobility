Code for paper "Extended Bayesian endemic-epidemic models
for incorporating mobility data into COVID-19 
forecasting" by Dirk Douwes-Schultz, Shuo Sun, Alexandra M. Schmidt and Erica Moodie.

Hennepin_data.RData - data for Hennepin County

king_county_data.RData - data for King County

Miami_data.RData - data for Miami County

NY_data.RData - data for New York County 

final_model_king_county.R - code for fitting the final model (underlined in Table 1) for King County.
Note that the final model for the other counties can be run by slightly modifying this code.
